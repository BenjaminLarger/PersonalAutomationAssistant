Automatically extract meeting details from Gmail and add them to your Apple Calendar

Processing your emails...

Setup Instructions:

  1. Create a .env file with your API keys (see .env.example)
  2. Set up Google OAuth credentials for Gmail access
  3. Configure CalDAV settings for Apple Calendar integration
  4. Label emails with meeting invitations as "meetings" in Gmail
  5. Run the automation to extract and save meeting details

Required Environment Variables:

  • OPENAI_API_KEY - Your OpenAI API key
  • GOOGLE_CLIENT_ID - Google OAuth client ID
  • GOOGLE_CLIENT_SECRET - Google OAuth client secret
  • CALDAV_URL - Apple iCloud CalDAV URL
  • CALDAV_USERNAME - Your Apple ID
  • CALDAV_PASSWORD - App-specific password
